# pushbullet-api
API built around Pushbullet for pushing notifications to devices

```json
Sample Request JSON
{
    "authCode": "********",
	  "deviceName": "David - Phone",
	  "action": "pushLink",
	  "linkName": "Pushbullet",
	  "linkUrl": "https://docs.pushbullet.com",
	  "linkBody": "Pushbullet Docs"
}
```
